# AI Text Composer for WordPress (AI-TC-WP)

AI Text Composer for WordPress is a plugin that automatically generates high-quality, SEO-optimized content for WordPress websites. Given a topic, keyword, or phrase, it generates a comprehensive, well-structured, and SEO-optimized article. The article is suitable for posting on a WordPress website. It includes an introduction, body, and conclusion. The plugin uses a `Search` capability to gather information about the topic if necessary.

## Installation

1. Upload the plugin files to the `/wp-content/plugins/ai-tc-wp` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress.
3. Use the Settings->AI-TC-WP screen to configure the plugin.

## Usage

1. Go to the AI-TC-WP settings page.
2. Enter your desired title, placeholder, and select the length (Short, Medium, Long) for your article.
3. Click on the 'Generate' button.
4. The plugin will generate a high-quality, SEO-optimized article for you.

## Files

The project includes the following files:

- ai-tc-wp.php: The main plugin file.
- includes/ai-tc-wp-settings.php: Contains the settings of the plugin.
- includes/ai-tc-wp-generator.php: Contains the generator of the plugin.
- includes/ai-tc-wp-search.php: Contains the search functionality of the plugin.
- includes/ai-tc-wp-article-structure.php: Contains the article structure of the plugin.
- admin/ai-tc-wp-admin.php: Contains the admin-specific functionality of the plugin.
- public/ai-tc-wp-public.php: Contains the public-facing functionality of the plugin.
- assets/css/ai-tc-wp-style.css: Contains the CSS for the plugin.
- assets/js/ai-tc-wp-script.js: Contains the JavaScript for the plugin.
- readme.txt: This file.
